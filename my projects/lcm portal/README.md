# College Learning Management System (LMS)

A comprehensive, modern Learning Management System designed for colleges and universities. This web-based platform provides students, instructors, and administrators with all the tools they need for effective online learning and course management.

## 🚀 Features

### For Students
- **Course Management**: View enrolled courses, access materials, and track progress
- **Assignment Tracking**: Submit assignments, view deadlines, and monitor submission status
- **Grade Analytics**: View detailed grade reports and performance analytics
- **Dashboard**: Personalized dashboard with course overview and recent activity
- **Mobile Responsive**: Access your courses from any device

### For Instructors
- **Course Creation**: Create and manage courses with materials and resources
- **Assignment Management**: Create assignments, set deadlines, and grade submissions
- **Student Tracking**: Monitor student progress and engagement
- **Grade Management**: Record and manage student grades
- **Communication**: Interact with students through the platform

### For Administrators
- **User Management**: Manage student and instructor accounts
- **System Overview**: Monitor platform usage and performance
- **Course Oversight**: Oversee all courses and academic activities
- **Reporting**: Generate reports on academic performance and engagement

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with modern design principles
- **Icons**: Font Awesome 6.0
- **Fonts**: Google Fonts (Inter)
- **Responsive Design**: Mobile-first approach
- **Local Storage**: For data persistence

## 📦 Installation & Setup

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Node.js (optional, for development server)

### Quick Start

1. **Clone or Download** the project files
2. **Open** `index.html` in your web browser
3. **Start using** the LMS immediately!

### Development Setup (Optional)

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start development server**:
   ```bash
   npm run dev
   ```

3. **Open** your browser to `http://localhost:3000`
